## 🔴 立即可做（本週）

### 1. 🔐 /admin 管理員驗證 (安全性 P0)

**問題**：目前 `/admin` 頁面依賴隱藏入口，亦缺乏強制的後端驗證，知道網址即可存取。

**實作建議**：
1.  **Firebase Auth Claims**：為特定 Email (如 `akai@school.edu.tw`) 設定 `admin: true` 的 Custom Claim。
2.  **路由保護**：
    ```tsx
    <Route path="/admin" element={
      <AdminRoute>
        <AnalyticsDashboard />
      </AdminRoute>
    } />
    ```
3.  **Firestore 規則**：鎖定 `/stats` 與 `/users` 集合讀取權限。
    ```javascript
    allow read: if request.auth.token.admin == true;
    ```

### 2. 🧪 測試覆蓋率 50% (品質 P0)

**當前狀態**：~35%
**目標**：補足核心邏輯測試

**優先測試項目**：
- `useAuth`: 登入/登出/狀態變化 (Mock Firebase)
- `useFirestore`: 資料讀取與寫入錯誤處理
- `CategoryFilter`: 分類篩選邏輯
- `ToolCard`: 點擊、收藏互動測試

## 🟡 短期目標（1-2 週）

### 3. 🖼️ 評論圖片與多媒體功能 (互動性 P1)

**應用場景**：學生上傳作業成果、老師分享使用截圖。

**技術方案**：
- **Firebase Storage**：建立 `/comments` 資料夾。
- **Image Resizer**：上傳前客戶端壓縮 (max 1920px)。
- **Lightbox**：點擊評論圖片放大預覽。

### 4. 🔔 PWA 推播通知 (回訪率 P1)

**功能**：
- 當有人回覆評論時通知。
- 當新工具上線時通知。
- 每日/每週精選工具推薦。

**技術方案**：
- Service Worker `push` 若事件監聽。
- Firestore 用戶 Token 儲存。
- Firebase Cloud Messaging (FCM) 發送。

## 🟢 中期目標（1-4 個月）— AI 創新與個人化

### 5. 🤖 AI 教案生成器 (Gemini API 整合)

**概念**：老師輸入「教學主題」與「年級」，AI 自動推薦適合的工具並生成 40 分鐘教案。

**流程**：
1.  輸入：「國小三年級 分數加減法」
2.  Call Gemini API (Flash 1.5)
3.  AI 回傳 JSON：
    -   **導入 (5min)**: 使用「趣味分數」工具引起動機。
    -   **發展 (25min)**: 使用「數學練習器」進行分組競賽。
    -   **總結 (10min)**: 學生分享。
4.  前端渲染成漂亮的教案卡片，可下載 PDF。

### 6. 👤 個人化學習歷程 (Profile)

**功能**：
- **成就展示櫃**：展示已獲得的徽章與等級。
- **學習雷達圖**：分析使用工具的類型分佈 (如：偏重「邏輯」或「語言」)。
- **收藏夾分類**：允許使用者建立多個收藏清單 (如：「數學課用」、「英文課用」)。

1.  **Import Alias 路徑優化**：全面改用 `@/components`, `@/hooks` 避免 `../../../`。
