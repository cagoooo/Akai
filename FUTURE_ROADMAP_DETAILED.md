## 🔴 立即可做（本週）

### 1. 🔐 /admin 管理員驗證 (安全性 P0)
**實作細節**：
- 使用 Firebase Authentication 結合 **Custom Claims**。
- 前端使用 React Router Guard (或 wouter matcher) 阻斷非法存取。
- 後端 Firestore Security Rules 需嚴格校驗 `request.auth.token.admin == true`。

### 2. 🩺 智慧工具連結巡檢系統 (健全性 P0)
**技術方案**：
- **GitHub Actions 排程**：每週一凌晨執行 Node.js 巡檢腳本。
- **巡檢邏輯**：對 `data.ts` 中的所有 URL 執行非同步 `fetch` (HEAD request)，檢測 404/500/Timeout。
- **回報機制**：若有連結失效，自動發送內容至 **LINE Notify** 或 **Discord Webhook**，讓阿凱老師能第一時間修復。

### 3. 🧪 單元測試覆蓋率與 Regression 監控 (品質 P0)
**重點項目**：
- 使用 **Vitest** 針對 `useAuth`, `useToolTracking`, `searchLogic` 進行 Mock 測試。
- 確保在 GitHub Actions 的 CI 流程中自動執行 `npm run test`。

## 🟡 短期目標（1-2 週）— 體驗與深度互動

### 4. 📂 協作工具清單 (Tool Playlists) (社群 P1)
**功能細節**：
- 使用者可點擊工具卡片上的「➕」按鈕，將工具加入「我的教學包」。
- **持久化**：使用 Firebase Firestore 存儲清單 JSON。
- **分享**：生成如 `/playlist/:id` 的短網址，點擊後可直接看到該老師精選的 5-10 個工具。
- **場景**：提供「開學第一週新手包」、「資優班邏輯訓練包」等官方推薦清單。

### 5. 🎨 節慶動態 UI (UX P1)
**功能**：在特定節慶時，背景自動更換為動態櫻花、星星或積雪效果（使用 HTML5 Canvas 或 CSS Particles）。

## 🟢 中期目標（1-4 個月）— AI 強化與個人化助手

### 6. 🔍 AI 語意搜尋與 RAG 推薦 (AI P1)
**技術亮點**：
- **方案**：將 50+ 個工具的標題與長篇詳細描述進行 **Text Embedding (Gemini API)**。
- **搜尋體驗**：使用者搜尋「我想要玩大風吹那種感覺的遊戲」，AI 能根據語意關聯性回傳「互動性遊戲」與「抽籤工具」等結果，而非僅匹配關鍵字。
- **推薦引擎**：根據用戶過去點擊記錄，主動推薦「除了這款，其他老師也常搭配使用...」的組合。

### 7. 🤖 自動教案 PDF 生成器
**技術路徑**：結合 Gemini 1.5 PRO，串接 Markdown 轉 PDF 套件，讓老師一鍵產出結構完整的教案筆記。

## 🔵 長期願景（3-6 個月）— 平台進化與生態系

### 8. 🎮 遊戲化班級營運 (Gamification)
**概念**：建立班級排行榜，學生使用工具累積的「經營點數」可以兌換虛擬獎勵或徽章。

### 9. 📱 跨端整合版本
**方案**：將 PWA 使用 Capacitor 打包成原生 iOS/Android App，進駐 App Store / Play Store 擴大接觸面。

## 🔧 持續演進的技術架構建議

1.  **動態載入優化**：針對非即時需要的組件（如後台與複雜圖表）使用 `React.lazy` 與 `Skeleton`，維持首頁載入在 1 秒內。
2.  **CDN 加速**：靜態預覽圖（PNG/WebP）移至 Cloudflare R2 或 Firebase Storage CDN，減輕主伺服器負擔。
3.  **i18n 多語言架構**：雖然目前以繁體中文為主，但建立好 `i18next` 架構有利於未來向國際教育市場推進。
