## 🔍 如何快速找到需要的工具？

### 方法一：搜尋功能 ⭐ 推薦

在首頁上方的搜尋欄輸入關鍵字，系統會自動搜尋：

- **工具名稱**：如「評語」、「計時器」
- **工具描述**：如「學生表現」、「課堂管理」  
- **標籤關鍵字**：如「AI」、「數學」、「遊戲」

#### 搜尋範例

| 想找什麼？ | 可以輸入 |
|------------|----------|
| AI 相關工具 | `AI`、`人工智慧` |
| 數學教學 | `數學`、`九九乘法` |
| 遊戲類工具 | `遊戲`、`闖關` |
| QR Code 工具 | `QR`、`二維碼` |
| 英文教學 | `英文`、`翻譯` |
| 學生評語 | `評語`、`評量` |

### 方法二：分類篩選

點擊分類標籤（如 🎮 趣味遊戲），即可篩選該類別的所有工具。

### 方法三：標籤 Chips

每個工具卡片下方都有 **標籤 chips**（如 `#AI` `#教學` `#互動`），點擊可快速了解工具特色。

## 💡 常用功能說明

### ❤️ 收藏工具

看到喜歡的工具？點擊卡片右上角的 **愛心按鈕** 加入收藏，下次就能快速找到！

### 🕐 最近使用

平台會自動記錄您最近使用的工具（最多 10 筆），方便快速存取常用工具。

### 📱 手機也能用

平台採用響應式設計（RWD），手機、平板、電腦都能順暢使用。還可以加到主畫面當作 App 使用！

### ⌨️ 鍵盤快捷鍵

| 按鍵 | 功能 |
|------|------|
| `/` | 快速跳到搜尋欄 |
| `Esc` | 清除搜尋 |
| `↑` `↓` | 瀏覽工具卡片 |
| `Enter` | 開啟選中的工具 |
| `F` | 收藏/取消收藏 |
| `?` | 顯示快捷鍵說明 |

## 📲 安裝到手機主畫面

### iPhone / iPad
1. 用 Safari 開啟平台網址
2. 點擊下方「分享」按鈕 (↑)
3. 選擇「加入主畫面」
4. 點擊「新增」

### Android
1. 用 Chrome 開啟平台網址
2. 點擊右上角「⋮」選單
3. 選擇「加入主畫面」
4. 點擊「新增」
